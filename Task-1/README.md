# 🚀 TechNova - Responsive Business Website

A modern and fully responsive multi-page business website built using **HTML5** and **CSS3** as part of the ApexPlanet Web Development Internship (Task 1).

## 📌 Features

* Responsive Navigation Bar
* Hero Section
* About Section
* Services Page
* Portfolio Gallery
* Blog Page
* Contact Form
* Google Maps Integration
* Testimonials Section
* Statistics Section
* Call-to-Action Section
* Responsive Design for Mobile, Tablet & Desktop
* Hover Effects & CSS Animations

## 🛠️ Technologies Used

* HTML5
* CSS3
* Flexbox
* CSS Grid
* Google Fonts
* Font Awesome
